## 트리 (Tree) 기출 핵심

### 트리 순회 — 시험에 가장 많이 나옴
```
전위 순회 (Preorder)  : 루트 → 왼쪽 → 오른쪽
중위 순회 (Inorder)   : 왼쪽 → 루트 → 오른쪽
후위 순회 (Postorder) : 왼쪽 → 오른쪽 → 루트
```

### 암기법
```
전위 = 루트가 제일 먼저 (Pre = 먼저)
중위 = 루트가 가운데
후위 = 루트가 제일 나중 (Post = 나중)
```

### 기출 문제 유형
```
트리가 주어지고 순회 결과 순서 맞추기
→ 그림 보고 직접 순서 추적하는 연습 필요
```

## 정렬 (Sorting) 기출 핵심

### 시간복잡도 — 반드시 암기
| 정렬 | 시간복잡도 | 기출 포인트 |
|------|-----------|------------|
| 버블 정렬 | O(n²) | 인접 원소 비교/교환 |
| 선택 정렬 | O(n²) | 최솟값 찾아 맨 앞으로 |
| 삽입 정렬 | O(n²) | 적절한 위치에 삽입 |
| 퀵 정렬 | O(n log n) | 피벗 기준 분할 |
| 병합 정렬 | O(n log n) | 분할 후 합병 |
| 힙 정렬 | O(n log n) | 힙 구조 활용 |

### 암기법
```
O(n²)      → 버블, 선택, 삽입 (BSI)
O(n log n) → 퀵, 병합, 힙 (QMH)
```

### 기출 문제 유형
```
1. 정렬 과정 중간 단계 보여주고 어떤 정렬인지 맞추기
2. 시간복잡도 맞추기
3. 특정 정렬의 특징 고르기
```
