# 2026-06-27 Windows & Unix

## Windows

### 주요 특징
- **GUI (그래픽 사용자 인터페이스)** : 아이콘·마우스 기반 직관적 조작
- **선점형 멀티태스킹 (Preemptive Multitasking)** : 운영체제가 각 작업의 CPU 사용 시간을 제어 → 한 프로그램 오류가 전체 시스템에 영향 주는 것을 방지
- **PnP (Plug & Play)** : 하드웨어 자동 인식·설치
- **OLE (Object Linking & Embedding)** : 응용 프로그램 간 객체 연결·삽입
- **단일 사용자 (Single User)** : 한 번에 한 사용자가 사용
- **255자 긴 파일 이름 지원**

### 자주 쓰는 단축키
| 단축키 | 기능 |
|--------|------|
| Alt + Tab | 실행 중인 프로그램 전환 |
| Alt + F4 | 현재 창/프로그램 종료 |
| Ctrl + Esc | 시작 메뉴 열기 |
| Win + D | 바탕화면 보기 |
| Win + E | 파일 탐색기 열기 |
| F2 | 이름 바꾸기 |
| F5 | 새로 고침 |

## Unix

### 주요 특징
- **시분할 시스템 (Time Sharing)** 기반
- **다중 사용자(Multi-user) · 다중 작업(Multi-tasking)** 지원
- **이식성(Portability)** : 대부분 **C언어**로 작성되어 다른 기종으로 이식 용이
- **계층적(트리 구조) 파일 시스템**
- **호환성·확장성** 우수
- 네트워크 기능 내장

### Unix 구성 (3계층)
| 구성 요소 | 역할 |
|-----------|------|
| **커널 (Kernel)** | Unix의 핵심. 항상 메모리에 상주하며 하드웨어 보호, 자원 관리(프로세스·메모리·입출력·파일 시스템) 담당 |
| **쉘 (Shell)** | **명령어 해석기**. 사용자 명령을 받아 커널에 전달. 종류: Bourne Shell(sh), C Shell(csh), Korn Shell(ksh) |
| **유틸리티 (Utility)** | 사용자가 자주 쓰는 프로그램 (DOS의 외부 명령어와 유사) |

### 주요 명령어
| 명령어 | 기능 |
|--------|------|
| `ls` | 디렉터리 내용 목록 출력 |
| `cd` | 디렉터리 이동 |
| `pwd` | 현재 작업 디렉터리 경로 출력 |
| `cp` | 파일 복사 |
| `mv` | 파일 이동 / 이름 변경 |
| `rm` | 파일 삭제 |
| `mkdir` | 디렉터리 생성 |
| `cat` | 파일 내용 출력 |
| `chmod` | 파일 접근 권한 변경 |
| `chown` | 파일 소유자 변경 |
| `ps` | 실행 중인 프로세스 확인 |
| `kill` | 프로세스 강제 종료 |
| `fork` | 새로운 프로세스 생성 |
