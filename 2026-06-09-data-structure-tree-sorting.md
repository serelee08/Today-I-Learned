## 자료구조 (Data Structure)

### 선형 자료구조
- **배열 (Array)**: 같은 타입 데이터를 연속된 메모리에 저장
- **스택 (Stack)**: LIFO (Last In First Out) — 마지막에 넣은 것이 먼저 나옴
- **큐 (Queue)**: FIFO (First In First Out) — 먼저 넣은 것이 먼저 나옴
- **연결 리스트 (Linked List)**: 노드가 포인터로 연결된 구조

### 비선형 자료구조
- **트리 (Tree)**: 계층적 구조
- **그래프 (Graph)**: 노드와 간선으로 연결된 구조

---

## 트리 (Tree)

### 기본 용어
- 루트(Root): 최상위 노드
- 부모(Parent): 상위 노드
- 자식(Child): 하위 노드
- 리프(Leaf): 자식 없는 최하위 노드
- 깊이(Depth): 루트부터 해당 노드까지 거리

### 이진 트리 (Binary Tree)
- 자식 노드가 최대 2개인 트리
- 왼쪽 자식 < 부모 < 오른쪽 자식

### 트리 순회 방법
- 전위 순회 (Preorder): 루트 → 왼쪽 → 오른쪽
- 중위 순회 (Inorder): 왼쪽 → 루트 → 오른쪽
- 후위 순회 (Postorder): 왼쪽 → 오른쪽 → 루트

## 정렬 (Sorting)

| 정렬 | 평균 시간복잡도 | 특징 |
|------|--------------|------|
| 버블 정렬 | O(n²) | 인접한 두 원소 비교/교환 |
| 선택 정렬 | O(n²) | 최솟값 찾아 앞으로 이동 |
| 삽입 정렬 | O(n²) | 적절한 위치에 삽입 |
| 퀵 정렬 | O(n log n) | 피벗 기준 분할 |
| 병합 정렬 | O(n log n) | 분할 후 합병 |
