## 1. 프로그래밍 언어 (개편 후 최대 비중)

### 1-1. 코드 실행 결과 추적 (Python / Java / C)
이번 개편의 핵심. 변수·연산자·제어문·반복문·함수·배열을 따라가며 **최종 출력값**을 손으로 추적하는 유형.

**자릿수 분해 (나머지/몫 연산) — 단골 패턴**
```c
int n = 1234;
int sum = 0;
while (n > 0) {
    sum += n % 10;   // 마지막 자리 추출 (4 → 3 → 2 → 1)
    n = n / 10;      // 자리 한 칸 제거
}
// sum = 1 + 2 + 3 + 4 = 10
```
핵심: `% 10` = 마지막 자리, `/ 10` = 자리 밀기.

**Python 추적 시 주의점**
- `/` (실수 나눗셈) vs `//` (정수 나눗셈) 구분
- 들여쓰기로 블록 결정 → 반복/조건 범위 착각 주의
- 리스트 슬라이싱 `a[1:3]`, `range(start, stop, step)`의 stop 미포함

**Java 추적 시 주의점**
- 정수/정수 나눗셈은 소수점 버림 (`7/2 == 3`)
- `++i` vs `i++` (전위/후위 증감) 평가 시점 차이
- 형 변환(int ↔ double) 시 값 손실

### 1-2. 제어 구조
- 조건문 (if / else if / switch)
- 반복문 (for / while / do-while) 및 break·continue
- 함수: 매개변수 전달, 반환값, (재귀 호출 추적)

### 1-3. HTML / CSS 기초
- 출제 수준: 명령어 서술을 읽고 **해당 태그/속성을 고르는** 정도
- 기본 태그(`<a>` `<img>` `<table>` `<div>`), CSS 선택자·속성 개념 정도만 정리

## 2. 응용SW기초기술

### 2-1. 운영체제
- 역할: 자원·프로세스 관리, 사용자 인터페이스
- 평가 기준: 처리량↑, 응답시간↓, 신뢰도↑, 사용가능도↑
- 프로세스 상태: 생성 → 준비 → 실행 → 대기 → 종료
- 스케줄링: 선점형(SRT, RR) vs 비선점형(FCFS, SJF, 우선순위)

### 2-2. Windows / Unix·Linux
- Windows: GUI, 선점형 멀티태스킹, 파일시스템 FAT/FAT32/**NTFS**
- Unix·Linux: **커널**(자원 관리) + **쉘**(명령어 해석기), C로 작성되어 이식성 높음
- 명령어: `ls` `cd` `cp` `mv` `rm` `cat` `grep` `chmod` `ps` `kill`
- **파일 권한**: rwx (읽기4/쓰기2/실행1). 예) `chmod 755` = rwxr-xr-x

### 2-3. 자료구조 / 정렬·검색
- 배열(임의접근 O(1)), 스택(LIFO), 큐(FIFO)
- 트리 순회: 전위(R→L→R) / 중위(L→R→R) / 후위(L→R→R)
- 정렬: 버블·선택·삽입·퀵·병합 (퀵·병합 평균 O(n log n))
- 검색: 순차 O(n), **이진검색 O(log n)** — 정렬된 데이터에만
