# 2026-06-16 — UI/UX 설계 (정보처리기능사)

## 오늘 공부한 것
정보처리기능사 시험 대비 UI/UX 단원 학습.

## 1. UI vs UX
- UI (User Interface): 사용자가 보고 조작하는 화면·요소 (버튼, 색상, 레이아웃)
- UX (User Experience): 사용자가 서비스를 쓰며 느끼는 전체 경험 (만족도, 편리함, 흐름)
- UI는 UX의 일부. 좋은 UI가 좋은 UX로 이어진다.

## 2. UI 설계 원칙 4가지
1. 직관성 — 누구나 쉽게 이해하고 사용
2. 유효성 — 사용자의 목적을 정확히 달성
3. 학습성 — 쉽게 배우고 익힐 수 있음
4. 유연성 — 사용자 요구 수용, 오류 최소화

## 3. UI 유형
- CLI: 명령어 텍스트 입력
- GUI: 그래픽(아이콘·버튼) 기반
- NUI: 터치·음성 등 자연스러운 동작
- VUI: 음성 명령 기반

## 4. UI 설계 도구
- 와이어프레임: 화면 뼈대를 선으로 표현
- 목업: 와이어프레임보다 실제에 가까운 정적 화면
- 프로토타입: 실제 동작하는 시제품
- 스토리보드: 화면 흐름 + 설명 문서
- 정밀도 순서: 와이어프레임 < 목업 < 프로토타입

## 5. 시험 포인트
- UI 설계 원칙 4가지 (직관성·유효성·학습성·유연성) 암기
- 와이어프레임 / 목업 / 프로토타입 차이
- UI 유형 약어 (CLI / GUI / NUI / VUI)
